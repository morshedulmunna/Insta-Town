Javascript is a synchronous single-threaded High level interpreted Programming language.JavaScript was created to communicate with the browser.It is run by the browser's V8 engine. Everything inside JavaScript is Execution Context. We can compare Execution Context with a box.There are 2 components inside the e-box. The first is memory and the second is code.Everything inside the memory is in the form of key: value like age : 20.

This stores everything inside the memory. Many people call memory a variable environment.A code component is a place where code is executed in exactly one line.Code is also called Thread of Execution in the same way that Memory is called Variable Environment.JavaScript is a synchronous single-threaded language.JavaScript is executed after reading each line of code from top to bottom, which we call synchronous single-threaded.

There is not much difference between Local Storage and Session Storage but it is not that there is not at all. The capacity of Local Storage is 10 MB and the capacity of Session Storage is 5 MB.

Local Storage can be accessed from any window and Session Storage can be accessed from the same tab, not from another tab.Local Storage will not expire for life unless the browser is uninstalled and Session Storage will expire as soon as you close that tab. for local storage developer can use localStorage.setItem('kye', 'value') for store data on local storage. for session strorage Developer can use sessionStorage.setItem('kye', 'value') for store data on session Storage.
